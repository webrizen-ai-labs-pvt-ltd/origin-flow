# Contributing to Origin Flow

**We do not accept external contributions, pull requests, or forks for this repository.**

This is a proprietary codebase owned by **Webrizen AI Labs Pvt Ltd.** It is hosted publicly strictly due to platform constraints (e.g., Vercel deployment requirements), not as an open-source project.

## Security Vulnerabilities

If you happen to find a security vulnerability, please do **NOT** open a public issue. Instead, please report it to our team privately.

Thank you for understanding!
